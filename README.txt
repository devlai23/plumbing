Loyalty program for Mochinut, allowing users to become members of the program and accumulate points for their purchases. They will then be able to redeem points for free products. In addition, in the back-end, the owner of the store will be able to gather analytics on loyalty customers.

This project uses HTML and CSS for the front-end website.
It also uses a MySQL database to manage customer information.
Python and Flask are used in the back-end to run the web server and communicate with the database.
